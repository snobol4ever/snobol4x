## What changed in emit.c (sprint stmt-fn)

- `trampoline_mode` flag — set by `-trampoline` CLI arg
- `emit_goto_target()` — emits `return block_X` instead of `goto _L_X` when flag set
- `emit_goto()` — emits `return (void*)_tramp_next_N` for fall-through
- All three inline `goto _SNO_NEXT_` spots in `emit_stmt` — trampoline-aware
- `emit_trampoline_program()` — new function:
  - Each main-level stmt → `static void *stmt_N(void) { ... }`
  - Sentinel pattern: `static char _tramp_sentinel_N; #define _tramp_next_N ...`
